    • Introduction

    PHP is a language typically used to make the hosting server generate a dynamic output - an output that is potentially different each time a browser requests a page.

    Some things to know about PHP, like any scripting lanaguge, is that it has a particular set of syntax rules that lets the processor know it's time to read PHP.

    The opening and closing tags for PHP starts with the less-than question-mark php symbol and ends when it sees the question-make greater-than symbol.

    <?php
     //Everything in between the php open and close tags will need to follow php syntax
    ?>

    Another basic syntax for PHP is the use of semi-colons to end all php commands. See the hello world example just below for reference.

    Lastly, regarding basic syntax for PHP, all variable declarations must start with the dollar sign symbol.

    Variables

    PHP makes variables simple as the type does not matter.

    Like javascript, you can assign strings, numerics and arrays to the same variable name.

    $stringVar = "Yue Chen"; //String assignment
    $numberVar = 25; //Integer assignment
    $numberVar = 22.5; //Double assignment
    $arrayVar = array("Laptop", "Phone", "MP3 Player"); //Array assignment

    Things to take into consideration, is variable naming conventions, see the list below:

  • Variable names must start with a letter of the alphabet or the underscore character.
  • Every character preceding the first can only contain alphabet characters, numbers or underscores.
  • Variable names cannot contain spaces; if there are multiple words for the variable name, use underscore for reability.
  • Variable names are case sensitive so $hello and $Hello will be seen as two different variables.

      echo "You cannot afford this gift!";

    Objects

    Objects are like functions, reusuable bits of code, but instead, a class or template used continously.

    A class in this case is a composite of code and data, which you will instantiate if you want to use it.

    The data assoicated with these instantiated objects are called properties and the functions of classes are called methods.

    When writing classes, its best to write you methods in a way where you data are not visible to anyone but the class functions.
